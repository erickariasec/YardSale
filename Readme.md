## Installation :electric_plug:
- Clone yardsales:

  - HTTPS:
```
git clone https://github.com/erickariasec/YardSale.git
```

  - SSH:
``` 
git clone git@github.com:erickariasec/YardSale.git
```

- Install dependencies:
```
npm install
```

- Local yardsale deploy:
```
npm run start
```
